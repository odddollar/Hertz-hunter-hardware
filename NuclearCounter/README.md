# NuclearCounter

Custom PCB for the [Hertz Hunter](https://github.com/odddollar/Hertz-Hunter) firmware.

<div align="center">
    <img src="./Images/Example.jpg" alt="Example" width="40%" />
</div>

### Features

- Fully integrated hardware
- Battery charging circuitry
- Single USB-C port for charging and data connection
- 18650 battery slot
- 1.3" OLED screen

### Parts included

- PCB
- ESP32-C3 Super Mini
- RX5808
- SMA connector
- 18650 battery slot
- 1.3" OLED screen
- Buttons
- Buzzer

### Parts not included

- Case
- 18650 battery
- SMA antenna
